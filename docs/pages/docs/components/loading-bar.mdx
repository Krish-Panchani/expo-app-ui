# Loading Bar

An animated top loading bar component for showing loading states

## Installation

```bash
npx expo-app-ui add loading-bar
```

## Usage

```tsx
import LoadingBar from "@/components/ui/loading-bar";

<LoadingBar color="#007AFF" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| color | string | '#007AFF' | Loading bar color |

## Examples

### Standalone

```tsx
<LoadingBar color="#007AFF" />
```

### With Context (Recommended)

```tsx
// See top-loading-bar-context documentation
```

## Features

- Smooth infinite animation
- Customizable color
- Uses react-native-reanimated
- Positioned at top of screen
